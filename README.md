# Overview

The Blockchain Explorer SSR web application build on NextJS framework.


## Installation

1. Clone the project.

2. Install the required dependencies.

```
npm install
```

3. Create `.env` in root directory.

4. Start application.

```shell
npm run dev
```

5. Visit http://localhost:3000/ in your browser.

# test
